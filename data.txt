Spring Dat JPA - модуль фреймворка Spring предназначенный для работы с данными
основанный на стандарте JPA.В Качестве реализации стандарта JPA модуль Spring
Data JPA использует бибилотеку Hibernate.

В рамках стандарта JPA для взаимодействия с сущностями испоьзуется
EntiyManager , в модуле Spring Data JPA принято работать с сущностями не на прямую
через EntityManager, а через дополнительные абстракции называемые репозиториями

Репозиторий - отдельный обьект описывающий для строго определенного типа
сущности содержающий в себе методы для работы с ее обьектом.В Spring Data JPA
представлюяют интерфейсы для которых не прописывается реализация руками
разработчика

Базовым интерфейсрм для создания репозитория является JpaRepository<T, ID> где
Т тип сущности с которой репозиториий должен работать а ID тип данных первичного
ключа сущности Т. Для каждой сущности должен быть описан отдельный репозиторий.

Методы доступные для любых производных JpaRepository<T,ID>:

* findById(Id id) : Optional<T> - производит поиск сущности со значением
первичного ключа id и возвращает найденную сущность обернутую в Optional либо
пустой обьект Optional.

* findAll() : List<T> - Возвращет список всех найденных сущностей.

* save(T obj) : void - создает сущность obj если она ранее находилось в базе
либо обновляет ее в обратном случае.

* delete(T obl) : void - производит удаление по сущности obj.

* deleteById(Id id) : void - производит удаление сущности по переданному
через параметры id.
